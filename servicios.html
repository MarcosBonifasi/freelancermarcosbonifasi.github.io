Aplicaciones de Escritorio

Desarrollo aplicaciones de escritorio con la tecnologia Java Stantard Edition.
Java es el lenguaje más demandado actualmente debido a que es un lenguaje robusto, altamente tipado y multiplataforma. He diseñado desarrollado aplicaciones que tienen conexion a Bases de Datos Relacionales con MySQL que actualmente es el gestor más usado a nivel mundial.
El diseño Front-End es desarrollado con JavaFX, el cual nos provee una enorme cantidad de paquetes para hacer de mejor manera las GUI. La reporteria la he manejado con iReport.
Si esta interasado en una App de Escritorio, ponte en contácto conmigo.
